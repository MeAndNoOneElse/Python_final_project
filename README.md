# SplitChek - Проект совместного финансового управления

Этот репозиторий содержит финальный проект по курсу Python, реализующий систему управления совместными расходами.

## Структура проекта

```
final_project/
├── programm/           # Основной код проекта
│   ├── server/         # FastAPI backend
│   ├── bot/            # Telegram бот
│   └── web/            # React фронтенд
├── report/             # Отчеты и документация
├── task/               # Задачи проекта
└── README.md           # Этот файл
```
